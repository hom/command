# nohup

> 后台执行命令，退出控制台继续执行

1. `zsh`执行

```bash
nohup node index &
```

2. 提示

```bash
zsh: you have running jobs
```

3. 解决办法

```bash
#在命令结尾添加!
nohup node index &!
#或者给相关的任务id添加disown
disown %1
```
